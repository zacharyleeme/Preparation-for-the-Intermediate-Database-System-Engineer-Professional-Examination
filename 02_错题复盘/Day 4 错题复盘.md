## 二、错题明细

### 第7题

**题目：**  
关于 `COUNT(*)` 与 `COUNT(列名)` 的说法，正确的是（D ）。

A. 两者都只统计非空值  
B. `COUNT(*)` 统计总行数，`COUNT(列名)`统计该列非空值个数  
C. `COUNT(*)` 不能用于整表统计  
D. `COUNT(列名)`会统计空值

**题目括号内作答答案：** D  
**答案解析区标准答案：** B

**正确答案解析：**  
本题考查聚合函数中最常见的基础区别：

- `COUNT(*)`：统计查询结果中的**总行数**
- `COUNT(列名)`：统计该列中**非空值**的个数

因此，正确说法应为 **B**，而不是 D。  
D 错在把 `COUNT(列名)` 误判成“会统计空值”，这与 SQL 的实际规则相反。

**你为什么做错：**  
你把 `COUNT(列名)` 的统计范围理解错了，误以为它会把空值也一起统计进去。  
但在 SQL 中，`COUNT(列名)` 只统计该列中**不为 NULL 的记录**；如果该列为空值，则不会计入结果。

**正确理解应为：**
- `COUNT(*)`：看“有多少行”
- `COUNT(列名)`：看“该列有多少个非空值”
- 判断这类题时，先区分“统计对象是整行，还是某一列的非空值”

**对应知识点：**
- 聚合函数 `COUNT()` 的基本含义
- `COUNT(*)` 与 `COUNT(列名)` 的区别
- 空值 `NULL` 对统计结果的影响

**纠偏记忆：**
- 星号 `*`：不盯某一列，统计总行数
- 列名：只数该列里“有值”的记录
- 记忆口令：**`COUNT(*)` 数行，`COUNT(列名)` 数非空列值**

**建议复习动作：**
1. 重新对比记忆：`COUNT(*)`、`COUNT(列名)`、`COUNT(DISTINCT 列名)` 三种常见统计口径
2. 再做一遍第6题、第7题、第28题，连起来判断“总行数 / 非空值个数 / 字段含义”
3. 以后看到 `COUNT(列名)` 题，先问自己一句：**这道题统计的是总记录数，还是该列非空记录数？**

## 三、本次练习暴露出的关键易错点

### 1. `COUNT(*)` 和 `COUNT(列名)` 容易混成同一种统计口径
这次唯一错误虽然只有 1 题，但它很典型。  
很多题会用“总人数”“非空记录数”“某字段个数”混淆你，如果不先判断统计对象，就容易选错。

要牢牢记住：

- `COUNT(*)`：统计总行数
- `COUNT(列名)`：统计该列非空值个数
- 遇到空值时，两者结果可能不同
