## 七、对备考规划和后续出题的影响

### 7.1 真题优先题源扩展

后续生成每日练习题、专项练习题和模拟题时，题源优先级可调整为：

1. `05_真题库/文本化补证_2016-2019/` 中的 2016～2019 文本化真题。
2. `05_真题库` 中已可文本抽取的 2020～2023 真题。
3. 2024～2025 真题待 OCR 后再纳入。
4. 真题不足时，再基于真题考点进行改编。

### 7.2 下午题训练重点进一步确认

2016～2019 与 2020～2023 合并看，下午题长期高频主线进一步明确为：

- 数据库设计 / ER / 关系模式 / 主外键
- 函数依赖 / 候选码 / 范式 / 分解
- SQL DDL / 视图 / 查询 / 统计 / 约束
- 事务并发 / 隔离级别 / 一致性
- 日志恢复 / 检查点 / Undo / Redo
- 系统分析 / DFD / 业务流程理解

因此后续 Day 19 之后的新题应优先从上述真题主线抽取或改编。
