六、规范化（下午第一大核心）
1. 范式口诀（必背）
范式	消除
1NF	字段不可分
2NF	消部分依赖
3NF	消传递依赖
BCNF	决定因素必须是候选键
4NF	消多值依赖
七、规范化标准答题模板（重点）
题型：判断是否满足3NF
标准模板
关系模式不满足3NF。

因为存在传递依赖：

A→B
B→C

因此：
非主属性C传递依赖于主键A。

不满足BCNF

十二、CREATE TABLE 高频模板（必背）
CREATE TABLE SC(
    Sno CHAR(10),
    Cno CHAR(10),
    Grade INT,
    PRIMARY KEY(Sno,Cno),
    FOREIGN KEY(Sno)
        REFERENCES Student(Sno)
);
十三、GROUP BY 高频陷阱
原则（超高频）

SELECT 中：

只能出现：

GROUP BY字段
聚合函数
错误写法
SELECT dept,name,COUNT(*)
FROM student
GROUP BY dept;

name 不合法。

十四、EXISTS 高频模板
SELECT *
FROM Student S
WHERE EXISTS(
    SELECT *
    FROM SC
    WHERE SC.Sno=S.Sno
);
十五、JOIN 高频模板
SELECT *
FROM Student S
JOIN SC
ON S.Sno=SC.Sno;
