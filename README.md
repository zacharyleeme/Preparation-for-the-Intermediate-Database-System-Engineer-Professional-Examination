## 后续维护原则

1. 新增真题来源时，必须标注：
   - 来源站点
   - 来源类型
   - 可确认程度
   - 适用用途
2. 没有真实抓取证据，不把页面细节写成已验证事实
3. Day 2 起，生成新内容前必须先做前一天错题复盘
4. 在每日学习资料的“今日练习”部分，无论题型为何（包括但不限于选择题、判断题、简答题、计算题、案例分析题等），只要进入具体题目层，所有题目都必须增加一个更细一级的标题，统一命名为：`第X题`。该规则适用于后续每日新生成的学习资料，目的在于便于答题、引用、复盘和快速定位。
5. 后续新生成的每日学习资料与错题复盘，统一遵循 `学习资料与错题复盘生成框架.md` 中固化的结构模板、核查边界、前后日衔接规则与正式表述口径。
